## Run it on your machine

1. Clone the project

   ```sh
   git clone https://github.com/aznamle/brandcreative.git
   ```

1. Install the dependencies

   ```sh
   cd brandcreative
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
